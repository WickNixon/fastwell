## Auth & Onboarding

- 🔴 As a new user, I want to create an account with email and password so that I can access Fastwell.
- 🔴 As a new user, I want to see a password eye toggle so that I can check I'm typing correctly.
- 🔴 As a user, I want a "Forgot password?" link so that I can regain access if I forget my password.
- 🔴 As a user, I want a password reset email with a button so that I can reset without copying a long link.
- 🔴 As a member, I want to click my personal invite link so that I'm taken straight to a password creation screen.
- 🔴 As a member, I want to see the Fastwell logo fade in when I click my invite link so that my first impression feels premium and warm.
- 🔴 As a new user, I want to be asked my first name so that the app feels personal from the very first screen.
- 🔴 As a new user, I want to choose my menopause stage from simple tappable cards so that onboarding feels easy.
- 🔴 As a cycling user, I want to enter my cycle length so that the fasting planner maps to my hormonal rhythm.
- 🔴 As a new user, I want to choose my primary goal so that the app shows me what matters most to me first.
- 🔴 As a non-member, I want a 14-day free trial so that I can try Fastwell before paying.
- 🔴 As a non-member, I want a clear paywall on day 15 so that I can choose a plan and continue.
- 🔴 As a member, I want my email verified automatically from the invite link so that I never have to enter payment details.
- 🔴 As a subscriber, I want to manage my subscription from within the app so that I stay in control.
